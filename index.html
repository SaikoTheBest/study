مستندات کدهای C# برای امتحان

این صفحه شامل توضیحات متدها و مفاهیم استفاده شده در تصاویر کد برای کار با فایل‌ها، پوشه‌ها و فرم در C# می‌باشد.

۱. کار با فایل‌های متنی (File I/O)

این متدها برای خواندن و نوشتن متن در فایل‌ها استفاده می‌شوند و در فضای نام System.IO قرار دارند.

File.AppendAllText(string path, string contents)

توضیح: این متد یک رشته متنی را به انتهای یک فایل در مسیر مشخص شده اضافه می‌کند. اگر فایل وجود نداشته باشد، آن را ایجاد می‌کند.

// اطلاعات چند TextBox با فرمت خاص (جدا شده با تب) به انتهای فایل info.txt اضافه می‌شود.
string data = string.Format("{0}\t{1}\t{2}\t{3}\r\n", textBox1.Text, textBox2.Text, textBox3.Text, "");
File.AppendAllText("info.txt", data);

File.ReadAllText(string path)

توضیح: این متد تمام محتوای یک فایل متنی را خوانده و به صورت یک رشته (string) برمی‌گرداند.

// کل محتوای فایل info.txt را می‌خواند و در textBox5 نمایش می‌دهد.
textBox5.Text = File.ReadAllText("info.txt");

File.WriteAllText(string path, string contents)

توضیح: این متد یک رشته متنی را در یک فایل می‌نویسد. اگر فایل از قبل وجود داشته باشد، تمام محتوای قبلی آن را پاک کرده و محتوای جدید را جایگزین می‌کند.

// محتوای textBox1 را در فایلی که مسیرش در متغیر documentFilePath است، می‌نویسد.
File.WriteAllText(documentFilePath, textBox1.Text);

۲. کار با پوشه‌ها (Directory)

این متدها برای دسترسی به لیست فایل‌ها و پوشه‌ها داخل یک مسیر استفاده می‌شوند.

Directory.GetDirectories(string path)

توضیح: لیستی از نام تمام پوشه‌های موجود در یک مسیر مشخص شده را برمی‌گرداند.

// لیست تمام پوشه‌های درایو C را به listBox1 اضافه می‌کند.
string directory = @"C:\";
listBox1.Items.AddRange(Directory.GetDirectories(directory));

Directory.GetFiles(string path)

توضیح: لیستی از نام تمام فایل‌های موجود در یک مسیر مشخص شده را برمی‌گرداند.

// لیست تمام فایل‌های درایو C را به listBox1 اضافه می‌کند.
string directory = @"C:\";
listBox1.Items.AddRange(Directory.GetFiles(directory));

۳. مدیریت فرم و کنترل‌ها

این دستورات برای مدیریت خود فرم برنامه و کنترل‌های روی آن استفاده می‌شوند.

this.Close()

فرم فعلی برنامه را می‌بندد.

Control.Clear()

محتوای یک کنترل را پاک می‌کند (متن TextBox یا آیتم‌های ListBox).

⭐️ نکته مهم و تصحیح کد

در کد مربوط به باز کردن فایل (openToolStripMenuItem_Click) یک اشتباه رایج وجود دارد. کد به جای خواندن فایل، محتوای TextBox را در فایل می‌نویسد.

کد صحیح باید به شکل زیر باشد:

private void openToolStripMenuItem_Click(object sender, EventArgs e)
{
    if (openFileDialog1.ShowDialog() == DialogResult.OK)
    {
        // مسیر فایل انتخاب شده را در متغیر ذخیره می‌کنیم
        documentFilePath = openFileDialog1.FileName;
        // محتوای فایل را می‌خوانیم و در TextBox نمایش می‌دهیم
        textBox1.Text = File.ReadAllText(documentFilePath);
    }
}
